# Angular Grid Demo App

Live demo application for the Angular 18 High-Performance Grid.

## Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm start

# Build for production
npm run build

# Build for GitHub Pages
npm run build:gh-pages
```

## Demo Features

- **Load 100K rows** - Test with 100,000 rows
- **Load 500K rows** - Stress test with 500,000 rows
- **Toggle Grouping** - Enable/disable row grouping by department
- **Expand/Collapse** - Click group rows to expand/collapse
- **Live Stats** - Real-time render time display
